SHChat Installation


Installation Führt eine komplette Installation durch und legt die Datenbank an.
Alte Datensätze von SHChat werden unwiderruflich gelöscht.
Config Erzeugt nur eine neue Config Datei. Bei Umzug oder Update notwendig.
Update Führt ein Update der Datenbank auf eine neue Version durch,
sofern eine install/update.php Datei vorhanden ist.
