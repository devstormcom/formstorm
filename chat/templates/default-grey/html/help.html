1 Benutzer-Ränge
Es gibt im Chat fünf verschiedene Benutzer-Ränge, die zu bestimmten Rechten bemächtigen.
Benutzer Ganz unten steht der Standard-Benutzer ohne besondere Rechte. Er kann in nicht-moderierten Channels schreiben und alle unter 3 aufgelisteten Chat-Kommandos ausführen.
Voice (+) Der sogenannte "Voice-Status" ist temporär und nur für den Channel gültig, indem er verliehen wurde. Er verleiht das Recht, in einem moderierten Channel zu schreiben. Einen Benutzer mit diesem Status erkennt man an dem Plus-Symbol vor dem Namen.
Channel-Operator (@) Der Channel-Operator Status wird temporär vergeben und hält, wie der Voice-Status nur solange, bis der Channel verlassen wurde. Er ist ein Benutzer, hat aber zusätzlich ein paar Operator-Rechte und kann durch das At-Symbol vor dem Namen erkannt werden.
Operator (@) Der Operator ist permanenter Wächter über den gesamten Chat. Er trägt im Gegensatz zum Channel-Operator auch im Profil den Rang "Operator" mit sich.
Administrator ($) Der Administrator ist in den meisten Fällen der Chat-Betreiber oder jemand, der die volle Kontrolle darüber besitzt. Er kann sämtliche Befehle ausführen und ist unverwundbar gegen Operatorbefehle. Er kann auch als einziger permanente Operatoren ernennen und kann am Dollar-Symbol vor dem Namen erkannt werden.


2 Channel
Entsprechend den Benutzer-Rängen gibt es sogenannte "Channel-Flags", die bestimmte Benutzerrechte erfordern können.
Moderated/Moderiert (+) In einem moderierten Channel können nur Benutzer mit mindestens Voice-Status schreiben. Man erkennt diesen Channel an einem Plus-Symbol vor dem Namen.
Hidden/Versteckt (.) Ein versteckter Channel taucht in der Channel-Liste nicht auf, selbst wenn er existiert. Meistens handelt es sich hierbei um ein Separée und wird durch das Punkt-Symbol vor dem Namen erkannt.
Maximum (x/y) Der Operator oder Administrator kann festlegen, dass ein Channel nur eine maximale Anzahl an Benutzern fasst. Dies wird durch die Zahlen in Klammern hinter dem Channel-Namen gekennzeichnet. Wenn es kein maximum gibt, steht dort nur die Anzahl der Benutzer im Channel.


3 Chat-Kommandos:
Chat-Kommandos werden wie ganz normaler Text eingegeben und abgeschickt. Jeder Befehl startet immer mit einem Schrägstrich.
/w Name Text Flüstert einem Chatter einen Text zu.
/join Channel Betritt einen Chat-Channel (z.B. Geheim1234).
/sep Channel Öffnet und betritt einen temporären, versteckten Chat-Channel (z.B. Geheim1234). Sobald dieser Channel leer ist, wird er gelöscht.
/inv Name Läd einen Benutzer in den Channel ein.
/away {Grund} Versetzt den Benutzer in den "Away-Modus" und hinterlässt {optional} einen Grund. Durch erneute Eingabe (/away) wird der Modus aufgehoben.
/quit Verlässt den Chat.


4 Admin-Kommandos:
Admin-Kommandos können von Benutzern mit bestimmten Rängen ausgeführt werden. Sie werden genau wie Chat-Kommandos benutzt.
@ Channel Operator
/v Name Der Benutzer erhält für diesen Channel den "Voice" Status und steigt somit einen Rang auf. Benutzer mit diesem Status können in moderierten Channels schreiben. Der Status wird verloren, sobald der Channel verlassen wird.
/uv Name Der Benutzer verliert seinen "Voice" Status und steigt somit einen Rang ab.
/kick Name {Grund} Schmeißt einen Benutzer aus dem Chat. Er kann danach erneut einloggen.
@ Operator (kann sämtliche Channel-Operator Befehle ausführen)
/s Name {Zeit} Verbietet einem Benutzer für eine bestimmte Zeit zu schreiben oder hebt das Verbot auf.
/mod Stellt die Schreibrechte dieses Channels auf "moderiert" bzw. "nicht moderiert".
/hidden Versteckt oder zeigt diesen Channel wieder in der Channel Liste.
/limit Zahl Stellt das Benutzerlimit dieses Channels auf "Zahl" ein.
/chanop Name Der Benutzer "Name" erhält einen temporären "Operator" Status für diesen Channel, steigt somit im Rang auf und erhält alle Channel-Operator Rechte.
/dechanop Name Der Benutzer "Name" verliert den temporären "Operator" Status.
/ban Name {Grund} Verbannt einen Benutzer aus dem Chat. Er kann danach nicht mehr einloggen.
$ Administrator (kann sämtliche Operator Befehle ausführen)
/bc Text Schreibt einen Broadcast an alle Benutzer im Chat (Channelübergreifend).
/op Name Der Benutzer erhält den globalen "Operator" Status, steigt somit im Rang auf und erhält alle Operator Rechte.
/deop Name Der Benutzer verliert den globalen "Operator" Status.
/unban Name Entbannt einen Benutzer. Er kann danach wieder einloggen.
